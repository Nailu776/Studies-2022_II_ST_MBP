# 2022_II_ST_MBP 

Repository for a project on Secure programming methods (Metody bezpiecznego programowania - MBP).

Application features a scientific calculator written in OCaml and operated via console.
Program is able to calculate polynomials as well as draw them inside a console window via symbols.
